# xinput-gtk
xinput-gtk is a GTK3 frontend for Xorg XInput

## Dependencies
- `xorg-xinput`
- `gtkmm-3`

## Installation
```
git clone https://github.com/mehedirm6244/xinput-gtk
cd xinput-gtk
mkdir build; cd build
cmake ..
cmake --build . -j 4
```
